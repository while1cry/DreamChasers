## 模块简介

🎈 欢迎来到 DreamChasers-Client 模块！这里包含两个基于 Vue.js 开发的网页前端，用于支持 梦想驿站 和 惜别梦途 的功能。
💻 这两个前端都使用 Node.js 构建，使用 Pinia 进行状态管理，使用 Vue Router 进行路由管理。
